WhatsApp Chat Reader & Rizz Rater

A tool to analyze WhatsApp chat exports and rate your rizz.

Problem Statement

WhatsApp chat exports are plain text files that are difficult to read and analyze. I wanted to create a tool that could parse these files, display them in a user-friendly format, and provide interesting insights and analytics, including a fun "rizz-rating" feature.

Tech Stack

  • HTML, CSS, JavaScript
  • Python
  • Flask

Architecture

The application is a simple web app powered by a Python back-end. The user uploads their WhatsApp chat export, which is then processed by a Python script. The script parses the chat file, performs analysis, and generates a report. The results are then displayed on a web page.

My Role

I was the sole developer for this project. I designed and built both the front-end and the back-end, including the chat parsing and analysis logic.

Key Challenges

  • Parsing the complex and sometimes inconsistent format of WhatsApp chat exports.
  • Developing a "rizz-rating" algorithm that is both fun and somewhat accurate.
  • Creating a simple and intuitive user interface.

Lessons Learned

This project was a great way to practice my Python and web development skills. I learned a lot about text processing, data analysis, and building simple web applications with Flask.
