My Role

I was the sole developer, responsible for all aspects of the project, from UI/UX design to implementing the machine learning model for pose estimation and rep counting.

Key Challenges

  • Optimizing the PoseNet model for real-time performance on mobile devices.
  • Developing an accurate rep-counting algorithm that could handle variations in exercise speed and form.
  • Designing an intuitive user interface that provided clear feedback to the user.

Lessons Learned

This project was a deep dive into the practical application of machine learning on mobile devices. I learned a great deal about performance optimization, model quantization, and the challenges of building a real-time, AI-powered application. I also gained valuable experience in designing user-centric features and iterating on them based on feedback.
