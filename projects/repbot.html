AI workout tracker built in Flutter with real-time pose estimation, automated rep counting, and performance analytics.

Problem Statement

Weightlifters and fitness enthusiasts often struggle to accurately track their workout repetitions and maintain proper form. This can lead to inefficient training and potential injuries. RepBot aims to solve this by providing a hands-free, AI-powered workout tracker.

Tech Stack

  • Flutter
  • Firebase
  • Google ML Kit (Pose Detection)
  • Dart

Architecture

The application uses a simple yet effective architecture. The Flutter front-end captures video frames from the device camera and passes them to Google's ML Kit for pose detection. A custom algorithm then analyzes the pose landmarks to count repetitions and provide feedback on form. Firebase is used for user authentication and data storage.

My Role

I was the sole developer for this project. I was responsible for all aspects of development, from UI/UX design to implementing the core AI features and integrating with Firebase.

Key Challenges

  • Developing an accurate rep-counting algorithm that works for various exercises.
  • Optimizing the pose detection pipeline to run smoothly on mobile devices.
  • Designing an intuitive user interface that provides clear and concise feedback.

Lessons Learned

This project was a great learning experience in mobile app development, computer vision, and machine learning. I gained valuable experience in building complex AI-powered applications and optimizing their performance for mobile devices.
