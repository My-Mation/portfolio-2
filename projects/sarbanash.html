A 2D top-down shooter with a unique twist, built with Unity.

Problem Statement

I wanted to create a fast-paced and challenging 2D shooter that would test players' reflexes and strategic thinking. I also wanted to experiment with a unique gameplay mechanic to make the game stand out from other titles in the genre.

Tech Stack

  • Unity
  • C#

Architecture

The game is built with Unity and uses a simple entity-component-system (ECS) architecture. The game logic is written in C#, and the assets are all custom-made.

My Role

I was the sole developer for this project. I was responsible for all aspects of development, including game design, programming, art, and sound.

Key Challenges

  • Developing a unique and engaging gameplay mechanic.
  • Creating all the art and sound assets from scratch.
  • Optimizing the game for performance on a variety of devices.

Lessons Learned

This project was a great way to learn the ins and outs of game development with Unity. I gained valuable experience in game design, programming, and asset creation. I also learned a lot about the challenges of indie game development and the importance of perseverance.
